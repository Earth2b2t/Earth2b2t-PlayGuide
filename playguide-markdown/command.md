# Commands  

Open a menu
```
/menu
```

Found a nation.
```
/nation new <国名>
```

Invite player to nation.  
```
/nation invite <名前>
```

Accept invite.
```
/nation accept <名前>
```

Decline invite
```
/nation deny <名前>
```

Promote player.  
```
/nation promote <名前>
```

Demote player.
```
/nation demote <名前>
```
Show infomartion about the nation of Belonging
```
/nation info
```

leave the nation.
```
/nation leave
```

Kick citizen from nation
```
/nation kick
```

Deposit the SC to the national treasury  
```
/nation deposit <数値>
```

Expand the country.
```
/nation claim
```

Relinquish nation. 
```
/nation disclaim
```

Reset nation.  
```
/nation reset
```

Start the war.
```
/nation war <国王の名前>
```

Pay compensation.
```
/nation pay <数値>
```

Surrender to war.
```
/nation surrender
```

Execute the command being checked.
```
/nation confirm
```
